🧠 How Backtracking Works

STEP 01

Find Empty Cell

Scan left-to-right, top-to-bottom. Stop at the first cell with value 0. If none found, the puzzle is solved.

  const idx = board.indexOf(0);
  if (idx === -1)
    return true; // solved!

Try Digits 1-9

STEP 03

Validate Placement

A digit is valid only if it does not already appear in the same row, column, or 3x3 box.

function isValid(b, i, v) {
